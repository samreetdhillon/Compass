Welcome!

This is an Open Source GUI for Teaching and Implementing Computational Physics Problems. Here you will study the dynamics of a number of physical systems and predict their evolutionary behaviour using several numerical techniques. You will also learn how to write C++ programs for solving these problems.


Here's how to use this program:

1. Go to Let's Learn!
2. Select a problem to study.
3. Select a numerical method to use.
4. Enter the inputs.
5. Click on Run .
6. To downlaod your data (inputs, outputs, computed data, graphs), click on Download Data .
7. To learn how to code the problem, click on View Program Code and practice the given exercises in the code and test the results on the provided online compiler.
8. To learn about the physics behind the program, click on Documentatiom


So let the learning begin!
