Open Source GUI for Teaching and Implementing Computational Physics Problems

This is an Open Source GUI for Teaching and Implementing Computational Physics Problems. Here you can study the dynamics of a number of physical systems and predict their evolutionary behaviour using several numerical techniques. You can also learn how to write C++ programs for solving these problems.
